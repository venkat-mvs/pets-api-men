# Pet Api

--------------------

A MEN stack based api.

### How to start?

1) Clone this repo

```sh
> git clone <this-git-repo-link>
```

2) Install dependencies

```sh
> cd <repo-directory>
> npm install 
```

3) put env variables

```sh
>  echo -e 'PORT=8080\nCONNECTION_STRING="<YOUR-CONNECTION-STRING-HERE>"' >.env
```

4) Start project

```sh
> npm start
```
